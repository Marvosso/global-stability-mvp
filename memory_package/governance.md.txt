# Governance Doctrine - Global Stability MVP

- Procedural Objectivity: No moral/ideological judgments.
- Tiered Publishing Workflow:
  - High Confidence / Low Severity → Auto-publish (post-review flagged)
  - Medium Confidence → Single human review
  - Low Confidence / High Severity → Two independent reviews, mandatory "worst-case scenario" sign-off
- All changes to classification, severity, or confidence must include justification and timestamp.
